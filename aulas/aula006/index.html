Principais formatações

Negrito / Destaque

Um termo em negrito usado html é fácil de pôr, coloque entre <strong> e </strong> (Uma forma semântica)

Itálico / Ênfanse

Um termo em itálico usado html é fácil de pôr, coloque entre <i> e </i> (Uma forma não semântica)

Um termo em itálico usado html é fácil de pôr, coloque entre <em> e </em> (Uma forma semântica)

Texto marcado

Isso é uma parte importante no texto tanto que está marcado, usando a tag <mark> e </mark>
